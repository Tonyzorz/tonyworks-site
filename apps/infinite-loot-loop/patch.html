Release Log

Patch Notes

New content, balance changes, and fixes—organized so players can see what changed at a glance.

Developer note This timeline is maintained from the project development log and rewritten into player-facing highlights.
2026-07-16

Mobile services, encounters & result polish

  • Fixed iPhone touch-and-hold movement: hold an empty part of the map to move, drag to steer, and release to stop.
  • Holding near the character now fills the Encounter Gauge, letting players seek a battle without leaving their current spot.
  • UI buttons and open menus now correctly block movement without decorative HUD layers swallowing map touches.
  • Gold purchases now require Purchase / Cancel confirmation, and unaffordable item, character, and slot buttons are disabled.
  • Improved rewarded-ad recovery for revives and run-over gold, banner positioning between runs, and App Store purchase availability feedback.
  • Redesigned the premium Store, item-drop result card, and equipment-upgrade indicator for clearer visual hierarchy.
  • Corrected normal item drops to one shared 3% roll per victory, and made recommended equipment stand out with a dedicated BEST badge.
  • Improved mobile readability for EXP and permanent stats, added clear Set purchase feedback, and reduced the overworld character size by 10%.
  • Rebalanced permanent item bonuses to 1% on the first copy and 3% at full completion, with existing saves recalculated from owned items.
  • Added full-edge safe-area padding for notches and rounded phone corners, completed battle prompt translations, and standardized Store purchase-button heights.
  • Added an optional translucent arrow-control pad for held movement, with a persistent on/off toggle in Settings.
  • Gold, EXP, and ATK zone bonuses now use distinct colors, with an ultra-rare rainbow Gold x7 bonus.
  • After the first boss victory, the game makes a one-time request for Apple’s native review prompt.
  • Improved Forest Upper Gate and Ashen Gate doorway detection, restored the Volcanic-to-Void route in both directions, and focused Witch’s Peak entry on its top-left cave.
  • Achievements now have a distinctive colored tab deck for All, Milestones, Combat, Collection, Normal, and Hard categories.
  • Erase Data now clearly covers both local and cloud gameplay saves, safely retries an offline cloud deletion on the next online launch, and keeps restorable Store entitlements.
  • Added edge-aware camera framing so entrances at the top and bottom of each map stay visible above the Set controls and lower HUD.
2026-07-05

Stability & localization pass

  • Fixed a hard freeze caused by corrupted enemy EXP values; level-up processing is now capped and EXP/gold are modeled per expected level.
  • Comprehensive 11-language localization sweep with a new re-localizing text component (boss / character / item / map names and the stat guide).
  • Main-menu Store button fix and a new character nameplate banner.
2026-06-12 · Session 42

Balance & polish

  • Adventure Point economy reconciled: 30 AP base · 4–6 AP on death · +3 AP per boss. Death now keeps your level, EXP, and gold.
  • Achievement stat bonuses are now derived from the unlocked set (no inflation, syncs across devices).
  • Full Reset now wipes all permanent progress.
  • Forest doorways re-laid (FR_03/FR_04) with zone levels flipped; central anti-stuck spawn corridor added.
  • Battle panel forced full-screen with a run-over tap guard; live derived stats in the stat panel; equipment icon blur fix.
  • Biome BGM name-matching fix; several doorway/trigger retunes across Forest, Volcano, Desert and Underwater.
Content complete

The game so far

  • 5 worlds — Grassland, Forest, Volcano, Desert, Underwater.
  • 25 characters (23 gold-unlock + 2 premium: a drop-rate specialist and a gold specialist).
  • Hard Mode, grouped settings, and 11-language localization.
  • Cloud save deployed and live; real in-app purchases wired up.
