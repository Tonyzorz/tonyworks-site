The Battle Point (BP) loop

BP is your run currency — the loop that ties everything together.

  • You start each game with 30 BP (plus any permanent BP bonuses).
  • A normal win costs 1 BP.
  • Dying costs 4 BP (Normal) or 6 BP (Hard Mode).
  • Beating a boss refunds +3 BP for the current run.
  • Death does not reset your level, EXP, or gold — only a full game over (hitting 0 BP) does.
Tip: Because bosses give +3 BP, clearing bosses is how you extend a run indefinitely. Farming trash costs net BP; boss kills pay you back.

World tiers & progression

The game is organised into worlds, and each world is a difficulty tier. Monster level, stats, and loot all climb together as the tier rises, so the tier is shorthand for “how strong is this area.” In Normal mode:

  • T0 · Grassland — the starter world (~Lv 1–25).
  • T1 · Forest — ~Lv 25–100.
  • T2 · Volcanic / Lava — ~Lv 105–195.
  • T3 · Desert — ~Lv 205–380.
  • T4 · Underwater — the deepest Normal world (~Lv 400–560).

From Grassland (the hub) you branch west into Forest then Volcanic, east into Desert, and north — the docks into Underwater. See the World Map for the full layout.

Hard Mode is a second lap. After Normal you re-run the same worlds from level 1 against much tougher, fixed-stat monsters — so it feels a full tier harder even though the raw numbers overlap the late-Normal range. What gates you there is grinding the Hard “H” gear, not just bigger monster numbers.

How this wiki labels things: every monster and item stat is read straight from the game data, and each monster page lists the world(s) it appears in. The “T0…T4” above is simply the Normal-world order; Hard Mode reuses the same worlds.

Combat math worth knowing

  • Defense: damage reduction is min(0.80, DEF / (DEF + 100)) — capped at 80%. Stacking DEF has diminishing returns.
  • Level up: the EXP needed for the next level is currentLevel × 150.
  • Revive: once per battle — watch a short ad (or free if you've removed ads) to get back up where you fell. It's BP-neutral; the death cost was already paid.

Loot & drop rates

  • Each enemy has a drop table with per-item chances (most are a few percent). Boss uniques are ~1% in Normal, ~0.75% in Hard.
  • Duplicate falloff: the more copies of an item you already own, the sharply lower the chance of another — so early copies come fast, then taper off hard.
  • LUC adds drop chance (about +0.2% per point, capped around +2%). Drop-rate gear, set bonuses, and certain characters stack on top.
  • Boss drops are rare: about 1% in Normal and 0.75% in Hard Mode before bonuses.

Shops & sets

  • The shop pulls from all registered zones, not just your current map.
  • Items with a set name grant a set bonus (including bonus drop rate) when worn together — check an item's page for its set mates.

Characters

Characters apply stat multipliers while active and some grant a permanent owned bonus that applies in every run just for owning them (for example, drop-rate and gold-gain specialists). See the Characters page for the full roster.

Hard Mode

Hard Mode raises boss HP/ATK to per-tier targets and changes drops (some bosses have a distinct Hard Mode drop). Death costs more BP (6 vs 4), so it rewards clean, well-geared runs.
