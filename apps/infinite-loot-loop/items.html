Equipment strategy

How to judge an item instead of chasing one large number

Reviewed 18 July 2026 · Values may change with balance updates

Immediate strength and permanent value

Equipment first matters because of the stats and effects it provides during the current run. A weapon may raise attack enough to shorten battles, while armor and helmets can add the health or defense needed to survive a new region. Shoes and accessories may solve different problems, including agility, luck, economy, or utility effects.

Collected copies can also contribute permanent progress. The permanent amount is derived from the item's eligible base stats and copy milestone rather than granting the full equipped value forever. This makes a modest early item useful without allowing one common drop to overwhelm the entire economy. Open an item to see its copy table and distinguish equipped strength from banked collection strength.

A practical comparison order

  1. Check whether the item improves the stat that currently blocks your route.
  2. Compare useful effects, not rarity color alone.
  3. Confirm whether the item completes a set or competes for an occupied slot.
  4. Review its drop source before spending a long run farming it.
  5. Use the permanent-bonus table to understand collection value over time.

The catalog's comparison tool is best used with items from the same slot. A higher price or rarity does not automatically make an item better for every character, because character multipliers and current route threats change the value of each stat.
